SMQE - linha de UX profissional-didática

Objetivo
- Apresentar o sistema como software técnico de engenharia voltado ao ensino de medidas elétricas, medição e qualidade de energia.

Princípios
- Aparência profissional: linguagem visual sóbria, terminologia técnica, métricas claras e componentes consistentes.
- Clareza didática: cada tela deve ajudar o aluno a entender o que está sendo medido, como foi calculado e como interpretar o resultado.
- Progressão pedagógica: priorizar o fluxo Dados -> Medidas -> Simulação -> Fasores -> Qualidade -> Energia -> Metrologia -> Relatórios.
- Leitura orientada: gráficos e tabelas devem destacar unidade, referência nominal e limite normativo.
- Transparência: quando um recurso for demonstrativo, isso deve ficar explícito para o usuário.

Diretrizes de interface
- Usar títulos curtos e descrições técnicas com foco em aplicação prática.
- Posicionar notas explicativas próximas aos painéis mais importantes.
- Tratar normas e limites como contexto de interpretação, não só como decoração visual.
- Evitar telas que exijam conhecimento implícito sem orientar o estudante.

Critérios por módulo
- Dashboard: síntese operacional com leitura guiada dos principais indicadores.
- Simulação: reforçar relação entre parâmetros de entrada e grandezas calculadas.
- Fasores: destacar referência angular, sequência de fases e componentes simétricas.
- Qualidade: explicar THD, eventos, conformidade e critérios de interpretação.
